# Monitoramento e Sensores Inteligentes na Agropecuária 🌱

Projeto educacional que simula o monitoramento de sensores em uma fazenda, mostrando dados de **temperatura**, **umidade** e **qualidade do ar**. Ideal para feiras escolares, projetos do Agrinho ou demonstrações de tecnologia agrícola.
